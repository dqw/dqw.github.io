MacOS安装homebrew

五 25 一月 2013
By dqw

homebrew是macos下的包管理工具，可以用来安装开源软件，比如apache、mysql等等，类似Debian的apt。比早起的MacPorts要好用，官方网址。

安装方法

mkdir homebrew && curl -L https://github.com/mxcl/homebrew/tarball/master | tar xz --strip 1 -C homebrew

常用命令

#查找软件包
brew search python

#安装软件包
brew install python

#列出已安装的软件包
brew list

#删除软件包
brew remove wget

#查看软件包信息
brew info python

#更新brew
brew update

#建立链接
brew link python
