MacOS安装zim

Fri 25 January 2013
By dqw

zim是跨平台的桌面wiki软件，主要用来做一些笔记日记用，windows和linux都有现成的安装包，macos下只能用源码安装还是比较麻烦，特意记录一下。

1、安装Homebrew

2、安装python

brew install python

3、安装gobject

brew install pygobject

4、安装pygtk

#会自动安装依赖，gtk+等
brew install pygtk

5、链接到/usr/local/bin

brew link python

6、由于macos自带的python链接到/usr/bin，所以会优先于Homebrew安装的python，所以要改一下path。修改/etc/paths，把/usr/local/bin改到/usr/bin前边。

7、定义PYTHONPATH

vim ~/.bashrc
export PYTHONPATH=/usr/local/lib/python2.7/site-packages:$PYTHONPATH

8、下载并安装下载并安装，hicolor-icon-theme

#先解压缩
./configure
make && make install

9、下载zim源码包，然后运行zim.py
